File structure

Within the download you will find the following file structure and content, logically grouping common assets and providing both compiled and minified variations.

Once downloading, unzip the compressed folder to see the structure of eXo UX Guidelines (based on Bootstrap). You will see something like this:

  exo-ux-guidelines /
  ├── css/
  │   ├── bootstrap.css
  │   ├── bootstrap.min.css
  │   ├── exo-platform.css
  ├── js/
  │   ├── bootstrap.js
  │   ├── bootstrap.min.js
  ├── img/
  │   ├── icon_start.png
  │   ├── icon_pattern.png
  │   ├── exo-platform-icons
  └── README.md

All CSS and JavaScript files are based on (bootstrap.*), as well as compiled and minified CSS and JavaScript (bootstrap.min.*).

(exo-platform.css) is a customized Bootstrap CSS that is applied for eXo Platform, done by eXo Platform Team.

Please note that most JavaScript plugins require jQuery to be included. Besides, some free JavaScripts are also applied into eXo UX Guidelines to create dynamic samples.

Basic HTML template

eXo UX Guidelines has a basic HTML template provided by Bootstrap.

Typical HTML file:

  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>Bootstrap Template</title>
  5.   </head>
  6.   <body>
  7.     <h1>Hello, world!</h1>
  8.     <script src="http://code.jquery.com/jquery-latest.js"></script>
  9.   </body>
  10. </html>

To make this a Bootstrapped eXo Platform Style template, just include the appropriate CSS and JavaScript files of Bootstrap and the customized CSS that is provided by eXo Platform Team.

Please note that exo-platform.css is the Core.css file that is located in skin/css of eXoResource.war in the eXo Platform package. You can copy, rename and put it in the general css folder that contains Bootstrap CSS. 

  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>Bootstrap Template</title>
  5.     <!-- Bootstrap -->
  6.     <link href="css/bootstrap.css" rel="stylesheet">
    <!-- eXo Platform CSS -->
  7.     <link href="css/exo-platform.css" rel="stylesheet">
  8.   </head>
  9.   <body>
  10.     <h1>Hello, world!</h1>
  11.     <script src="http://code.jquery.com/jquery-latest.js"></script>
  12.     <script src="js/bootstrap.min.js"></script>
  13.   </body>
  14. </html>

And you're set! With those files added, you can start developing any site or application with eXo Platform Style based on Bootstrap.
