1.Download Source

Before downloading, be sure that you have basis knowleadge on HTML, JS and CSS. Notepad++,Text Editor, Quanta Plus, Sublime Text are useful code editors to modify the file content.

Get the original files for all CSS and JavaScript,along with a local copy of the docs by downloading the latest version directly from GitHub

Download UXP Pattern Website

2. File structure

Within the download you'll find the following file structure and contents, logically grouping common assets and providing both compiled and minified variations.

Once downloaded, unzip the compressed folder to see the structure of UXP Patterns Website (based on Bootstrap). You'll see something like this:

  exoplatform-uxp-patterns-website/
  ├── css/
  │   ├── bootstrap.css
  │   ├── bootstrap.min.css
  │   ├── plf4.css
  ├── js/
  │   ├── bootstrap.js
  │   ├── bootstrap.min.js
  ├── img/
  │   ├── glyphicons-halflings.png
  │   ├── glyphicons-halflings-white.png
  └── README.md

All CSS, JS files are based on (bootstrap.*), as well as compiled and minified CSS and JS (bootstrap.min.*).

(plf4.css) is the customized Bootstrap CSS that is applied for the Platform 4, done by The eXo Platform Team.

Please note that most JavaScript plugins require jQuery to be included. Besides, there some free javacripts are also applied in UXP Pattern Website to create some dynamic samples

3. What's included

UXP Patterns Website are completely based on Bootstrap that is customized by Platform 4 style with the main content below:

Pattern

Based on Twitter Bootstrap, we provide uxp patterns with Platform 4 style for specific functions or objects that can be applied in common to obtain the conistency among products

Component

Details for some basic components that are applied in UXP Pattern Website, provided by Twitter Bootstrap under Platform 4 style.

Style

Basic css for html elements that are applied in UXP Pattern Website.

Javascript

Similar to Components, these Javascript plugins are interactive components for things like tooltips, popovers, modals, and more.

4. Basic HTML template

UXP Patterns Website has basic HTML templated provided by Bootstrap

Typical HTML file:

  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>Bootstrap Template</title>
  5.   </head>
  6.   <body>
  7.     <h1>Hello, world!</h1>
  8.     <script src="http://code.jquery.com/jquery-latest.js"></script>
  9.   </body>
  10. </html>

To make this a Bootstrapped Platform 4 style template, just include the appropriate CSS and JS files of Bootstrap and the customized CSS that is provided by The Platform 4 Team:

Please note that the plf4.css is the Core.css file that is located in skin/css of eXoResource.war in the Platform 4 package. You can copy, rename and put it in the general css folder that contains Bootstrap CSS. 

  1. <!DOCTYPE html>
  2. <html>
  3.   <head>
  4.     <title>Bootstrap Template</title>
  5.     <!-- Bootstrap -->
  6.     <link href="css/bootstrap.css" rel="stylesheet">
    <!-- Platform 4 CSS -->
  7.     <link href="css/plf4.css" rel="stylesheet">
  8.   </head>
  9.   <body>
  10.     <h1>Hello, world!</h1>
  11.     <script src="http://code.jquery.com/jquery-latest.js"></script>
  12.     <script src="js/bootstrap.min.js"></script>
  13.   </body>
  14. </html>

And you're set! With those files added, you can begin to develop any site or application with Plform 4 Style based on Bootstrap.

What next?
