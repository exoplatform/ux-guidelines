How to change color style?

Thank to eXo Skin Customization feature, the color style can be changed flexibly.

The main color variables: @baseColor, @baseBackground, @primaryBackground, @primaryColor (in variables.less) can be modified to change all main colors of eXo Skin.

After re-buidling with varible changes, the eXo Skin will be generated new color style basically.

In advanced customziation, simply modify variables of the corresponding ui elements.

For example, dark blue theme can be generated with the valiable changes below:@primaryColor: #0ab5f5; @baseColor:#ffffff; @baseBackground:#333333; @primaryBackground:#222222.
