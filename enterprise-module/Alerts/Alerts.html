Overview

An alert message is a notification about specific error, information, warning or successful actions.
Alert messages are suitable for the notifications that do not force user to do any action such as clicking to close. This type of notification will be automatically closed after seconds based on its configuration.
Besides, the alert message style is good in case of attracting user's attention to specific notifications.

Warning

Alerting a situation that might cause a problem in the future.

Your password is not strong enough. 
<div class="alert alert-warning" id="">
    <i class="uiIconWarning"></i>Your password is not strong enough.
</div>

Success

Alerting successful, completed or done actions.

Question What is eXo? has been added successfully. 
<div class="alert alert-success" id="">
<i class="uiIconSuccess"></i>Question <a href="">What is eXo?</a> has been added successfully.
<

Information

Alerting useful and relevant, but not critical information. It lets users know something new.

There is a new comment on your question. What is eXo? 
<div class="alert alert-info" id="">
    <i class="uiIconInfo"></i>There is a new comment on your question. <a href="">What is eXo?</a>
</div>

Error

Alerting wrong actions that have been done.

Your email address is incorrect. Please try again! 
<div class="alert alert-error" id="">
    <i class="uiIconError"></i>Your email address is incorrect. Please try again!
</div>

Note: To display an alert with limited width following the alert content length, simple use code style="display: inline-block;"

Your email address is incorrect. Please try again! 
<div class="alert alert-error" id="" style="display: inline-block;">
    <i class="uiIconError"></i>Your email address is incorrect. Please try again!
</div>
