Basic tabs

Take a regular <ul> of links and add uiTabPane, uiTabNormal.

  • Properties
  • Settings

        <ul id="myTab" class="nav nav-tabs">
            <li class="active">
                <a href="#tab-home" data-toggle="tab">Home</a>
            </li>
            <li>
                <a href="#tab-properties" data-toggle="tab">Properties</a>
            </li>
            <li>
                <a href="#tab-settings" data-toggle="tab">Settings</a>

        </ul>
    </div>
