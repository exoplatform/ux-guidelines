Popup

A useful and nice way to display advanced information. How and when a popup is used with the eXo Platform style?

  • Overview
  • Popup size
  • Default popup
  • Popup with block tabs
  • Popup with border
  • Popup without button

Overview

Popup is a form which is displayed when users select an option or doing an action. It can be used to perform specific functions or simply notify necessary information to users.

When to use?

This is a useful and fast way to interact with a function and display advanced options over one element. In particular, a popup is often used in the following cases:
Data input or displayed are short.
Simple functions.
Common components used by different applications.
Content which can be logically grouped in different tabs.
Multiple functions which can be interacted at the current page.

Popup size

It is suggested the default width of popups should be 560px. The popup height will be auto-resized depending on content and height of different screens. A scroll bar is shown if content is longer than the display screen.
