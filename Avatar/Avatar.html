Avatar

Basic CSS for avatar, how to display avatar in different styles?

  • Style
  • Size

Style

Based on Bootstrap, there are some PLF 4 avatar styles. In detail:

Circle Border Avatar

Add class .avatarCircle to <img> element to have circle avatar style.

This style is useful in case you want to display a square picture as a circle one for specific goal. E.g: User avatar for the main activity sharing

<img src="..." class="avatarCircle">

Rounded Corner Avatar

Add class .avatarMedium to <a> element that includes <img> to have rounded corner avatar style.

This style is default one for avatar with different size.

<a="avatarMedium"><img src="..."></a>

Rounded Corner Avatar With Border

Add class .avatarLargeWithBorder to <img> element to have rounded corner avatar with border style.

It is used in some case to make the avatar look more clear and formal.

<a="avatarXLarge"><img src="..."></a>
