Label and Badges

Basic and special CSS for different label and badges style

  • Label
  • Badges

Label

Labels Markup Apply For
Default <span class="label">Default</span> Explanation label or priority information label
Success <span class="label label-success">Success</span> Label for success information such as success notification in a pop-up
Warning <span class="label label-warning">Warning</span> Label for warning information such as warning notification in a pop-up, or warning information before doing actions.
Important <span class="label label-important">Important</span> Label for important information such as error notification in a pop-up
Info <span class="label label-info">Info</span> Label for useful or neccessary information such as new updates notification...
Inverse <span class="label label-inverse">Inverse</span> In case inverse color is applied for a page or pop-up, inverse buttons can be applied correspondingly
